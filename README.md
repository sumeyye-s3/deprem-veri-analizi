# deprem-veri-analizi

# Türkiye Deprem Veri Analizi Projesi

Bu projede Python kullanarak Türkiye'deki depremler analiz edilmiştir.

## Kullanılan teknolojiler

- Python
- pandas
- matplotlib
- folium

## Yapılan analizler

- Deprem büyüklüğü analizi
- Deprem kategorileri analizi
- En büyük depremlerin tespiti
- Harita üzerinde deprem görselleştirmesi

## Harita çıktısı

deprem_haritasi.html dosyasında interaktif harita bulunmaktadır.

## Amaç

Veri analizi ve veri görselleştirme becerilerini geliştirmek.
